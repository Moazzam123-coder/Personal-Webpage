Project Overview

This Drones is built by me with collaboration of my college mates this drone costed us around 16k with every damage and other things that happened with it now our drone specialization is that it will work on speech recognition well the work in still on it is in the prototype stage it can listen and work with different commands like start motors,fly,fly higher <to increase the throttle >,max< for maximum throttle %>,land,etc

Hardware Used

  1. Bluetooth Module HC02
  2. X-frame
  3. 1000Kv Motors
  4. Plastic Propellers
  5. ESCs
  6. GPS Module
  7. PCB
  8. Power MOdule
  9. 1300 Mah Battery
  10. Arduino Uno
  11. Landing Gear
  12. MPU module
